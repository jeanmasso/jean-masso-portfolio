# Portfolio — Jean Masso

Ce repository contient le code source de mon portfolio professionnel.

## Structure

- `frontend/` : application Angular (V1 — frontend-only)
- `backend/` : futur backend API (prévu pour une version ultérieure)

## Version actuelle (V1)

La V1 est une application statique développée en Angular :

- Standalone components
- Routing + lazy loading
- Données pilotées par JSON
- Internationalisation FR/EN
- Widget live (intégration API météo)
- Formulaire de contact via Netlify Forms
- Déploiement sur Netlify

Aucun backend personnalisé n’est utilisé dans cette version.

## Évolution prévue (V2)

- Backend API dédié
- Gestion dynamique des contenus
- Rubrique “Projets”
- Architecture full-stack

## Déploiement

Le déploiement sera effectué sur Netlify (configuration via `netlify.toml` à la racine).
